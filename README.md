## Instrucciones para ejecutar el proyecto

### 1. Clonar el repositorio

```bash
git clone <url_del_repositorio>
```
## 2. Ingresa a la carpeta dentro del repositorio
```bash
cd ProyectoAmerinst\
```
## 3. Abrir VsCode
```bash
code .
```
## 4. Instalar composer y npm
```bash
composer install
npm install
```
## 5. Levanta el servidor
```bash
npm run dev
php artisan server
```
